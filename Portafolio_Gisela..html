// sobre-mí

Soy Gisela Astudillo, estudiante de Desarrollo Full Stack en Desafío Latam. Entré al mundo del desarrollo web porque me gusta crear interfaces que pasen de boceto a algo con lo que la gente puede interactuar.

Hasta ahora he trabajado principalmente con HTML, CSS, JavaScript y React para construir interfaces responsivas y funcionales, y actualmente estoy sumando SQL a mi stack para entender mejor el lado del backend. También tengo experiencia en diseño UX/UI, aplicando metodologías como Design Thinking para investigar usuarios y prototipar soluciones en Figma. Cada proyecto de este portafolio es un paso más en ese camino: desde páginas estáticas hasta interfaces con lógica e interactividad real, pasando por procesos completos de diseño centrado en el usuario.

Busco mi primera oportunidad como desarrolladora junior, donde pueda seguir aprendiendo en equipo y aportar desde el primer día.

HTML5 CSS3 JavaScript React Bootstrap Git / GitHub SQL (en progreso) Figma / UX-UI
